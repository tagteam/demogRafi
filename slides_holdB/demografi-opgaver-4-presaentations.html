Opgave 1

Beregn de summariske mortalitetsrater for alle 4 lande (Danmark, USA, Indonesien og Nigeria) i 2022. Angiv hvilken formel fra Kompendiet du har brugt. Tjek om du har tastet og regnet rigtigt ved at sammenligne med kolonne Crude Death rate som du nu tilføjer ved at bruge Custom Report Columns. Rapporter resultaterne som antal døde per 1000 personår. Hvad tænker du kunne være vigtige faktorer, som kan forklare forskellene i de summariske mortalitetsrater mellem de 4 lande?

Løsning

Link med info: https://www.census.gov/data-tools/demo/idb/#/table?COUNTRY_YEAR=2024&COUNTRY_YR_ANIM=2024&menu=tableViz&CCODE=DK,ID,NG,US&TABLE_YEARS=2022&TABLE_USE_RANGE=N&TABLE_USE_YEARS=Y&TABLE_STEP=1&TABLE_ADD_YEARS=2022&quickReports=CUSTOM&CUSTOM_COLS=POP,CDR,DEATHS

library(gt)
dat <- tibble(Land = c("Danmark", "Indonesien", "Nigeria", "USA"),
              Folketal = c(5920767,277329163,225082083,333287557),
              "Antal døde" = c(56373,1873021,1959083,3128620))
dat <- mutate(dat,"Summarisk mortalitetsrate (døde per 1000 personår)" = 1000*Døde/Folketal)
gt(dat) %>% 
  fmt_number(columns = "Summarisk mortalitetsrate (døde per 1000 personår)", decimals = 1, locale = "da") %>%
  fmt_number(columns = c("Antal døde", "Folketal"), decimals = 0, locale = "da")

Opgave 2

Find folketal og det samlede antal døde i Danmark i 2022 i statistikbanken (Alder i alt) og beregn den summariske mortalitetsrate. Sammenlign data og resultatet med tal, som du har fundet i den internationale database (Opgave 1). Hvad tænker du kunne være en forklaring på forskellen mellem tal fra statistikbanken og den internationale database? Find bagefter folketal og antal døde for alle aldre i statistikbanken og inddel dem i 3 aldersintervaller: 0-14, 15-64, 65+. Bestem risikotidens aldersfordeling og de aldersspecifikke mortalitetsrater. Angiv hvilke formler fra kompendiet du bruger.

Løsning

Alder i alt:

R <- hent_data("folk1a",alder = "Alder i alt",tid = "2022K3")
D <- hent_data("dod",alder = "Alder i alt",tid = 2022)
dat <- tibble(R = R$INDHOLD,Dod = D$INDHOLD)
dat <- mutate(dat, M = 1000*Dod/R)
dat %>% 
  rename("Risikotid" = R, "Antal døde" = Dod, "Summarisk mortalitetsrate (døde per 1000 personår)" = M) %>%
  gt() %>% 
  fmt_number(columns = "Summarisk mortalitetsrate (døde per 1000 personår)", decimals = 1, locale = "da") %>%
  fmt_number(columns = c("Antal døde", "Risikotid"), decimals = 0, locale = "da")

x <- hent_mortalitetsrate_data(breaks = c(-Inf, 14, 64, Inf),

                               tid = "2022",
                               label_one = "0-14",

# Aldersspecifikke mortalitetsrater i Danmark
x <- mutate(x, M = 1000*Dod/R)
# Aldersfordeling i alle 4 lande
idb <- hent_IDB_screenshot_data()
# Beregning af kompendiets formel K2.11 
idb <- left_join(idb,select(x,aldersinterval,M),by = "aldersinterval")
y <- group_by(idb,Land) %>% summarize(expected = sum(M/1000*Risikotid))
# Antal dødsfald fra Opgave 1 
y <- mutate(y,observed = c(56373,3128620,1959083,1873021))
y <- mutate(y,SMR = observed/expected)

Kommentarer

  • Antal dødsfald i Indonesien/Nigeria/USA i 2022 var 2,46 gange/3,3 gange/0,63 gange end hvad man vil forvente, hvis mortalitetsraterne var de samme som i Danmark i 2022.

  • Svagheder: Man kan bruge resultaterne til at sammenligne Danmark med de andre 3 lande, men ikke direkte til at sammenligne de andre 3 lande indbyrdes, fordi SMR fra de 3 lande med Danmark som standardbefolkning afhænger landenes aldersfordeling af risikotid. Tallene er ikke særlig præcise på grund af de brede aldersintervaller. Data fra den internationale database er ikke præcist; det kan vi se for Danmark, hvor tal fra statistikbanken afviger.
